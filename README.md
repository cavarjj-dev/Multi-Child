# Multi-Child Family Navigator

Interactive assessment tool by The Quiet Edge.

## Setup Tracking (Optional)

1. Create a Google Sheet with a tab named exactly: `Multi-Child`
2. Add headers: Timestamp | Number of Children | Children Details | Primary Challenge | Comparison Signs Count | Comparison Signs | Positive Signs | Scenario Selected | Resource Info
3. Go to Extensions → Apps Script, paste the tracking script
4. Deploy as Web App (Execute as: Me, Access: Anyone)
5. Copy the URL and replace `YOUR_GOOGLE_APPS_SCRIPT_URL_HERE` in index.html

## Deploy to GitHub Pages

1. Upload these files to your GitHub repository
2. Go to Settings → Pages → Deploy from main branch
